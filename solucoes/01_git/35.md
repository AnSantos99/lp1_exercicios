## Problema

35 - Antes de começares a usar Git LFS no teu PC e na tua conta de utilizador,
que comando é necessário executar?

## Soluções

### Solução 1

Antes de se começar a usar o Git LFS é necessário executar o seguinte comando:

```
$ git lfs install
```
