## Pergunta

3 - Dá uma vista de olhos na documentação da classe
[`Math`](https://docs.microsoft.com/dotnet/api/system.math)
(_namespace_ [`System`](https://docs.microsoft.com/dotnet/api/system)).
É possível instanciar objetos desta classe? Porquê?


## Soluções

### Solução 1

Não, os objetos da classe "Math" não podem ser instanciados pois apesar de a
classe em causa não ser "abstract" (que indica que a classe é suposta ser
apenas uma classe base e não é suposta ser instanciada por si só), esta classe
"Math" tem como assinatura: 

public static class Math

Public é o nível de acesso menos restrito de todos, logo, permite a classe ser
instanciada, no entanto, é devido a esta ser "static" que não é possível ser
instânciada pois não é possível criar variáveis da classe base com o uso de "new"
("Classe" "Nome de Objeto" = new "Método()") o que faz com que esta
não possa ser instanciada, tendo o utilizador de aceder aos membros da classe
através da própria classe.
