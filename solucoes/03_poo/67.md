## Problema

67 - Escreve um método `static` que inicialize e devolva um
[enumerável genérico](https://docs.microsoft.com/dotnet/api/system.collections.generic.ienumerable-1)
contendo _n_ cópias de um valor passado como parâmetro e tipo especificado como
argumento genérico.

> [Soluções](../solucoes/03_poo/67.md)
## Soluções

### Solução 1

```csharp
static IEnumerable<T> CreateEnumerable<T>(T ogObject, int copiesNumber)
{
    for (int i = 0; i < copiesNumber; i++)
    {
        yield return ogObject;
    }
}
```

### Solução 2

```csharp
static IEnumerable<T> CreateEnumerable<T>(T ogObject, int copiesNumber)
{
    List<T> tempList = new List<T>();
    for (int i = 0; i < copiesNumber; i++)
    {
        tempList.Add(ogObject);
    }

    IEnumerable<T> collectionT = tempList as IEnumerable<T>;
    return collectionT;
}
```
